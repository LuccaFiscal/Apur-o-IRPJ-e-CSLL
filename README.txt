# Apuração IRPJ & CSLL — Lucro Presumido (HTML único)

## Como usar
1. Faça o download do ZIP e extraia.
2. Abra `index.html` no navegador (duplo clique).
3. Preencha os campos. Para gerar um PDF, clique em **Baixar PDF** (usa a impressão do navegador).

## Publicar na internet
- **Vercel/Netlify:** arraste e solte a pasta (ou conecte um repositório com esses arquivos). 
- **GitHub Pages:** crie um repositório com esses arquivos e ative o Pages apontando para a branch principal.
- **Qualquer servidor estático:** suba `index.html` e `favicon.ico`.

## Observações
- Ao escolher **Comércio/Indústria**, as presunções mudam automaticamente para **8% (IRPJ)** e **12% (CSLL)**. 
- Os campos aceitam colagem com `.` e `,` (ex: `84.000,00`). 
- O PDF gera **capa** com Empresa/Competência e destaca em **negrito** os valores de recolhimento.
